1. setting-->keymap 可以配置自己想要的快捷键
    删除当前行  ctrl + d
    复制当前行  ctrl + alt + 向下光标
    补全代码   atl + /
    多行注释   ctrl + /
    添加注释和取消注释 ctrl + /
    导入需要的类,配置auto import
    快速格式化代码 ctrl + shift + L
    快速运行程序 alt + R
    生成构造方法 alt + insert
    查看一个类的层级关系 ctrl + H
    定位方法  ctrl + B
    自动分配变量名,通常在后面.var
