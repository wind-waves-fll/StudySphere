1. Class类对象不是new出来的，而是系统创建的
2. 对于某个类的Class类对象，在内存中只有一份，因此类只加载一次
3. 每个类的实例都会记得自己是由哪个Class实例所生成。
4. 通过Class可以完整的得到一个类的完整结构，通过一系列API
5. Class对象是存放在堆中的
6. 类的字节码二进制数据，是放在方法区的，有的地方称为类的元数据
