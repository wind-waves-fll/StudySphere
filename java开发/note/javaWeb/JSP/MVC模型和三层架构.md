## MVC模式：
MCV是一种分层开发的模式，其中：
1. M：Model,业务模型，处理业务
2. V：View,视图，页面展示
3. C：Controller,控制器，处理请求，调用模型和试图
![alt text](assets/MVC模型和三层架构/image.png)
MCV好处：
- 职责单一，互不影响
- 有利于分工合作
- 有利于组件重用
